## 实现说明

- `ConfigLoader` 仅暴露 `LoadFromFile(const std::filesystem::path&, Config*)`
- JSON 解析完全位于 `src/config/config_loader.cpp`
- 使用 `nlohmann/json` 将 JSON 转换为强类型 `Config`
- 统一返回 `common::Status`
- 缺失字段、未知字段、字段类型错误、枚举非法值、范围错误都会返回明确字段路径和原因
- `std::filesystem::path` 用于目录字段，未调用 Windows/Linux 平台 API
- `config` 仅依赖 `common`、C++17 与 `nlohmann/json`

## 验证规则

已实现并校验：

- `node_id` 非空
- `ip_address` 非空
- `initial_members` 非空
- 当前 `node_id` 必须出现在 `initial_members`
- 成员 `node_id` 唯一
- 成员地址唯一
  - 实现为 `ip_address:raft_port:metadata_port:store_control_port` 组合唯一
- 所有端口必须在 `1-65535`
- 当前节点 `raft_port`、`metadata_port`、`store_control_port` 互不冲突
- `election_timeout_min_ms <= election_timeout_max_ms`
- `heartbeat_interval_ms < election_timeout_min_ms`
- `heartbeat_interval_ms`、`election_timeout_min_ms`、`election_timeout_max_ms`、`rpc_timeout_ms`、`queue_capacity`、`worker_count`、`max_message_size`、`log_batch_size`、`store_heartbeat_timeout_ms`、`failure_detection_interval_ms`、`task_poll_limit` 必须为正数
- `data_directory` 与 `snapshot_directory` 不得为空
- 成员角色仅允许 `VOTER` 或 `LEARNER`
- 日志级别仅允许 `TRACE`、`DEBUG`、`INFO`、`WARN`、`ERROR`、`CRITICAL`
- 根对象和成员对象的未知字段返回明确错误
- 必填字段缺失返回明确错误

## 手动失败场景验证

已验证以下失败场景，均返回明确错误：

- 当前节点不在成员列表：
  - `INVALID_ARGUMENT: node_id must exist in initial_members: node-x`
- 重复 `node_id`：
  - `INVALID_ARGUMENT: initial_members contains duplicate node_id: node-1`
- 重复地址：
  - `INVALID_ARGUMENT: initial_members contains duplicate address: 127.0.0.1:7101:7201:7301`
- 非法端口：
  - `INVALID_ARGUMENT: root.raft_port: must be in range 1-65535`
- 当前节点服务端口冲突：
  - `INVALID_ARGUMENT: raft_port, metadata_port, and store_control_port must be distinct`
- `heartbeat_interval_ms` 不小于 `election_timeout_min_ms`：
  - `INVALID_ARGUMENT: heartbeat_interval_ms must be less than election_timeout_min_ms`
- `queue_capacity` 为 0：
  - `INVALID_ARGUMENT: root.queue_capacity: must be in range 1-18446744073709551615`
- `max_message_size` 为 0：
  - `INVALID_ARGUMENT: root.max_message_size: must be in range 1-18446744073709551615`
- 非法成员角色：
  - `INVALID_ARGUMENT: initial_members[3].role: must be one of: VOTER, LEARNER`
- 非法日志级别：
  - `INVALID_ARGUMENT: root.log_level: must be one of: TRACE, DEBUG, INFO, WARN, ERROR, CRITICAL`
- 缺少必填字段：
  - `INVALID_ARGUMENT: root.rpc_timeout_ms: missing required field`
- 未知字段：
  - `INVALID_ARGUMENT: root.unexpected_field: unknown field`
